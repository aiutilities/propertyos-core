# ADR-008: Single Tenant Before SaaS

## Status

Accepted

## Context

PropertyOS must support self-hosting and data ownership.

Starting with complex SaaS multi-tenancy can slow down the foundation.

## Decision

PropertyOS will support a simple single-tenant deployment model first.

SaaS-style multi-tenancy can be designed later.

## Consequences

- Faster first usable version.
- Easier self-hosting.
- Future migration path to multi-tenancy must be considered carefully.
