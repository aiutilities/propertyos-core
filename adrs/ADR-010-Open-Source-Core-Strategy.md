# ADR-010: Open Source Core Strategy

## Status

Accepted

## Context

PropertyOS is positioned as an open-source operating system for properties, communities, and shared spaces.

## Decision

PropertyOS Core will follow an open-source-first strategy.

Commercial opportunities may exist around hosting, premium plugins, marketplace services, support, and enterprise features.

## Consequences

- Community trust improves.
- Ecosystem growth becomes possible.
- Licensing decisions must be handled carefully before public release.
