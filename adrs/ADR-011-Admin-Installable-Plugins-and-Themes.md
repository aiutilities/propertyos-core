# ADR-011: Admin Installable Plugins and Themes

## Status

Accepted

## Context

PropertyOS is inspired by WordPress.

Administrators should not need server access for normal plugin or theme management.

## Decision

PropertyOS admin panel must support plugin and theme installation, activation, deactivation, update, rollback, preview, and configuration.

## Consequences

- Better usability for non-technical admins.
- Plugin packaging, validation, and rollback become core requirements.
- Security review is mandatory before marketplace publishing.
